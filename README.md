## 🚀 Featured Projects

### 🤖 AI Document Summarizer
> `FastAPI` `Transformers (T5, BART)` `NLP` `Python`

Built a full-stack AI document summarization platform supporting **PDF, DOCX, and TXT** formats. Designed REST APIs with FastAPI and applied hierarchical + extractive NLP techniques to significantly improve summarization quality over baseline transformer outputs.

---

### 📈 LSTM-Based Portfolio Optimization
> `Python` `LSTM` `PyTorch` `Machine Learning` `Finance`

Developed an ML-driven portfolio optimization system combining **LSTM volatility forecasting** with **Modern Portfolio Theory (MPT)** for dynamic asset allocation. Achieved an **18% improvement in Sharpe Ratio** over traditional MPT — a strong, quantifiable result in the quant finance space.

---

### 🫀 Heart Sound Classification Using CNNs
> `Deep Learning` `CNN` `Signal Processing` `PyTorch`

Built a CNN-based classifier to detect normal vs. abnormal heart sounds using the **PhysioNet PCG dataset**. Extracted **MFCC and spectrogram features** to achieve high accuracy in identifying pathological cardiac conditions — a direct healthcare application of AI.
